# Lobo77
기본 룰 

1. 시작할 때 카드 5장을 받는다.

2. 돌아가면서 자신의 카드 중 한 장을 낸다.
그리고 카드를 다시 1장 받는다.

3. 낸 숫자의 합이 11의 배수,
또는 77 이상이 되면 게임에서 패배하게 된다.


세부 룰 

1. 카드들의 합은 공개되지 않으며 자신이 기억해야 한다.

2. 이 게임은 컴퓨터와 함께 진행되며, 컴퓨터가 내는 카드는 랜덤으로 정해진다.

3. 컴퓨터의 지능 상향을 위해 고른 카드와 그전 카드의 총합이 11의 배수가 되면 다시 선택하는
코드가 내장되어있다.
